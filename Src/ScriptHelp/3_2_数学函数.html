3.2 数学函数
  • double cos(double x)

返回x的余弦cos(x)值,x为弧度

  • double sin(double x)

返回x的正弦sin(x)值,x为弧度

  • double tan(double x)

返回x的正弦sin(x)值,x为弧度

  • double acos(double x)

返回x的反余弦cos-1(x)值,x为弧度

  • double asin(double x)

返回x的反正弦sin-1(x)值,x为弧度

  • double atan(double x)

返回x的反正切tan-1(x)值,x为弧度

  • double cosh(double x)

返回x的双曲余弦cosh(x)值,x为弧度

  • double sinh(double x)

返回x的双曲正弦sinh(x)值,x为弧度

  • double tanh(double x)

返回x的双曲正切tanh(x)值,x为弧度

  • double acosh(double x)

返回x的反双曲余弦值,x为弧度

  • double asinh(double x)

返回x的反双曲正弦值,x为弧度

  • double atanh(double x)

返回x的反双曲正切值,x为弧度

  • double log(double x)

返回log(x)值

  • double log2(double x)

返回log2(x)值

  • double log10(double x)

返回log10(x)值

  • double pow(double x, double y)

返回x的y次幂值，和(x ^ y)一样

  • double sqrt(double x)

返回x的开方值

  • double exp(double x)

返回以e为底的x次方值

  • double abs(double x)

返回绝对值

  • double sign(double x)

如果x<0，返回-1，如果x>0返回1,否则返回0

  • double rint(double x)

返回最近的整数

  • double if(double x, double y, double z)

如果x!=0，返回y值，否则返回z值

  • double min(double v1, double v2, …double vn)

返回给定参数值中的最小值，可以传递任意个参数。

  • double max (double v1, double v2, …double vn)

返回给定参数值中的最大值，可以传递任意个参数。

  • double sum (double v1, double v2, …double vn)

返回给定参数值的和，可以传递任意个参数。

  • double avg (double v1, double v2, …double vn)

返回给定参数值的平均值，可以传递任意个参数。
